GPR-Integrated Drone

Ground Penetrating Radar integrated with Aero360's Bhima logistics drone for non-contact subsurface mine detection.

Drones Engineering R&D

// overview

Overview

This project integrates a Ground Penetrating Radar (GPR) system with Bhima — a logistics drone developed by Aero360 — to detect metallic objects like mines beneath the soil surface without any physical contact. The system enables efficient wide-area scanning from the air, providing a safer alternative to ground-based detection methods.

The drone carries the GPR payload at a calibrated height, scanning the terrain below in real time while transmitting processed data for analysis.
